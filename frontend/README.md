
# Frontend – Ecommerce API + Stripe (modo test)

## Descripción
Este es el **frontend** del proyecto de examen técnico **Ecommerce con API Syscom + Stripe + Nodemailer**.  
Desarrollado con **Next.js**, **Tailwind CSS** y **TypeScript**, este cliente consume el backend del proyecto para mostrar productos, procesar pagos y visualizar el carrito de compras.
Este proyecto se desarrollo usando de base una plantilla ecommerce en Next.js, creditos para 
**NextMerce**
[![GitHub](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white) ](https://github.com/NextMerce/nextjs-ecommerce-template)

## Instalación

1. Clonar el repo y moverse a la carpeta frontend
```bash
git clone https://github.com/Jesus-Valverde/ecommerce-api-stripe-junior-jesus-valverde.git
cd frontend
```

2.  Instalar dependencias
```
npm install
```

3. En la carpeta frontend/, copiar el archivo .env y agregarle tus variables personales

```

cp .env.local.example .env.local

```

---

## Scripts

-   Iniciar servidor en modo desarrollo

npm run dev

-   Ejecutar tests (si los agregas)
